# ZeroTierCompanion使用方法


### 1.获取网络ID
在ZeroTier[官网](https://www.zerotier.com/)登录或注册账号，创建网络，就可以看到一个16位长度的网络ID了，例如 a7c8142af31857a1


### 2.获取TOKEN
登录官网后，右上角 Account---API Access Tokens---New Token,把生成的32字节的字符串复制下来,例如 q39SSzaOQAZ0XlPdP7SEOE6tc03sL7eA


### 3.使用ZeroTier伴侣
在手机上打开ZeroTier伴侣，输入 网络ID 和 TOKEN即可查看在线的设备


# 关于界面

以列表形式显示设备

每个列表元素显示的内容为设备状态、设备名称、局域网IP、设备公网IP。

其中设备状态用红色圆点表示，红色为离线，绿色为在线。
